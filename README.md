This project presents a laboratory-based comparative analysis of three network security monitoring and intrusion detection systems: Snort, Suricata, and Zeek.
The study was conducted in a controlled virtualized environment simulating a small home network. The goal was to evaluate how each system detects and interprets different types of network traffic, including ICMP requests, TCP connection attempts, UDP traffic, port scanning, and flood-like behavior.
The project focuses on differences in detection models (signature-based vs behavior-based), logging approaches, and the level of detail provided by each tool.
The results are based on isolated tests performed under identical network conditions to ensure consistency and reproducibility.
